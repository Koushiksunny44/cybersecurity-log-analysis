# Cybersecurity Log Analysis

## Project Overview
This project analyzes security login logs using Python and detects suspicious login activity based on failed authentication attempts.

The system identifies:
- Failed login attempts
- Suspicious IP addresses
- Login activity trends

## Technologies Used
- Python
- Pandas
- Matplotlib

## Features
- Log analysis
- Failed login detection
- Suspicious IP identification
- Data visualization

## Dataset
Sample simulated security log dataset containing:
- timestamps
- usernames
- IP addresses
- login status

## Project Structure

```bash
cybersecurity-log-analysis/
│
├── data/

```

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```

## Output
The project generates:
- Failed login analysis
- Suspicious IP detection
- Login visualization charts
